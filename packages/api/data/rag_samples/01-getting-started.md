# 시작하기 — 계정 만들기와 첫 로그인

my-agents 헬프센터에 오신 것을 환영합니다. 이 문서는 처음 가입한 사용자가 계정을
만들고 첫 로그인을 마치기까지의 과정을 안내합니다.

## 계정 생성
가입 페이지에서 이메일과 비밀번호를 입력하면 계정이 만들어집니다. 비밀번호는 최소
8자 이상이어야 하며, 영문과 숫자를 섞는 것을 권장합니다. 가입 직후에는 일반 사용자
권한이 부여되며, 관리 콘솔 접근은 슈퍼유저만 가능합니다.

## 첫 로그인
로그인은 이메일과 비밀번호로 진행합니다. 로그인에 성공하면 세션 쿠키가 발급되어 이후
요청에 자동으로 첨부됩니다. 비밀번호를 잊은 경우 로그인 화면의 "비밀번호 재설정"을 통해
이메일로 재설정 링크를 받을 수 있습니다.

## 다음 단계
로그인 후에는 에이전트 목록에서 미리 만들어진 에이전트와 대화해 볼 수 있습니다. 직접
에이전트를 만들고 싶다면 "에이전트 만들기" 문서를 참고하세요.
