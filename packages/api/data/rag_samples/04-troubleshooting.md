# 문제 해결 — 자주 묻는 오류와 대처

이 문서는 운영 중 자주 마주치는 오류와 그 대처법을 모았습니다.

## 로그인이 안 됩니다
세션 쿠키가 만료되었을 수 있습니다. 다시 로그인하면 새 쿠키가 발급됩니다. 그래도 안 되면
비밀번호 재설정을 시도하세요.

## 컬렉션을 만들 수 없습니다
임베딩 모델이 등록되어 있지 않으면 컬렉션 생성이 막힙니다. 프로바이더·모델 메뉴에서 임베딩
종류(kind=embedding) 모델을 먼저 등록하세요. 모델의 출력 차원이 저장소 차원과 다르면 생성이
차원 불일치로 거부됩니다.

## 문서 업로드가 error 상태가 됩니다
임베딩 서버에 연결되지 않거나, 모델의 출력 차원이 컬렉션 차원과 다를 때 발생합니다. 문서의
오류 메시지를 확인하고, 모델 점검(health) 기능으로 차원 일관성을 검사하세요.

## 검색 결과가 비어 있습니다
컬렉션에 적재된 문서가 없거나, 질의와 의미적으로 가까운 청크가 없을 때입니다. 관련 문서를
먼저 업로드하고, 질의를 더 구체적으로 바꿔 보세요.
