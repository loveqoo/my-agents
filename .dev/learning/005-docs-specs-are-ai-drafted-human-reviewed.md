# 005 — docs 스펙은 AI 초안 + 인간 검토

날짜: 2026-06-19
type: feedback

## 무엇을 배웠나
- `docs/`는 인간 영역이지만, **스펙은 인간이 직접 쓰기보다 AI가 초안을 쓰고 인간이 검토**하는 방식을 원한다.
- 사용자: "원래 docs는 내가 작성하는 공간인데, 내가 작성하는 것보다 네가 작성을 하고 내가 검토할 거야. 너가 초안을 도와준다고 생각하면 돼."

## 그래서 규칙
- `docs/spec/`은 **AI가 초안 작성 → 인간 검토**. (cf. [[002]] docs/는 인간 영역이라는 원칙의 정제)
- 그 외 `docs/`(adr 등)는 여전히 명령 없이 수정 금지.
- CLAUDE.md(Scaffolding·빠른 합의 규칙)에 반영함.

## 적용
- 스펙이 필요하면 먼저 질문으로 의도를 좁힌 뒤 **내가 초안을 docs/spec에 쓰고**, "검토해줘 + 빨간펜"으로 넘긴다.
- 결정은 인간이 한다(→ [[003]] 재촉하지 말 것).
